Compilação: O programa deve ser compilado com python3 trabalho_1.py

Execução:
Será perguntado ao utilizador se quer que os pontos sejam aleatórios (1) ou arbitrários(2):

-Se quiser que os pontos sejam aleatórios, deverá introduzir N, que é o número de pontos, e M, que é o intervalo das coordenadas (vão de [-M,M])

-Se quiser que os pontos sejam arbitrários, deverá introduzir N e cada ponto como dois números separados por um espaço e fazendo enter

-Deverá depois ver as imagens com os gráfico dos polígonos para cada uma das questões (feito com o matplotlib)
-As imagens serão geradas no mesmo diretório em que o programa for salvo

*É possivel que seja necessario instalar algumas bibliotecas caso sua versão do python3 não esteja atualizada

Resolução das questões: 1 à 5
